# Safety Model

resi is educational. It does not diagnose, prescribe, provide therapy, or serve as emergency care.

Risk categories include vaping, diabetes and habits, screen time, mental health, self-harm or crisis, substance use, eating/body image concerns, bullying, medical urgency, AI dependency, and misinformation exposure.

Severity routing:

- `LOW`: educational support.
- `MODERATE`: support, resource recommendation, and possible internal flag.
- `HIGH`: trusted-adult support and parent alert recommendation.
- `CRITICAL`: safety-first escalation language and parent alert workflow.

Parents do not see raw chat logs by default. Admin views use aggregate analytics only, with a minimum group threshold before demographic slices should be shown.
